# Training Planner - План тренировок

**Автор:** Галкаина Валерия

**Версия:** 1.0.0

## 📋 Описание проекта

Training Planner - это графическое приложение для планирования и отслеживания тренировок. Приложение позволяет добавлять, просматривать, фильтровать и удалять записи о тренировках, сохраняя данные в JSON файл.

### Основные возможности

- ✅ Добавление тренировок с указанием даты, типа и длительности
- ✅ Просмотр всех тренировок в удобной таблице
- ✅ Фильтрация по типу тренировки и по дате
- ✅ Удаление выбранных тренировок
- ✅ Автоматическое сохранение данных в JSON файл
- ✅ Загрузка сохраненных данных при запуске
- ✅ Валидация ввода данных (формат даты, положительная длительность)
- ✅ Удобный интерфейс с поддержкой клавиатурных сокращений

### Технологии

- Python 3.x
- Tkinter (графический интерфейс)
- JSON (хранение данных)
- Unittest (тестирование)
